## A starter kit for OpenGL development

To build the project, run ```cmake --build build```. This will create 
an executable "game" located at ```build/src/game```.

Additional resources on included libraries:<br>
    - OpenGL https://learnopengl.com/<br>
    - GLFW   https://www.glfw.org/<br>
    - CMake  https://cmake.org/cmake/help/latest/guide/tutorial/index.html<br>
    - glad   https://glad.dav1d.de/<br>
